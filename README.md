# Kubernetes en détail

## Une série interactive en 3 parties par Lighthouse Labs et Microsoft

Découvrez Kubernetes avec des professionnels de l’industrie chez Microsoft.

Avec les services Web modernes, les utilisateurs attendent de leurs applications qu’elles soient disponibles en tout temps. Les développeurs doivent donc déployer de nouvelles versions de ces applications plusieurs fois par jour. La conteneurisation avec Kubernetes permet d’empaqueter les logiciels pour répondre à ces objectifs, en assurant le lancement et la mise à jour simple et rapide des applications.
